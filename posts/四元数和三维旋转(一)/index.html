四元数和三维旋转(一)

本文最后更新于：2025年4月17日 晚上

由于使用欧拉角进行旋转时会存在万向锁的问题，同时，用欧拉角进行插值也不尽方便，而基于四元数的旋转既解决了万向锁的问题，又能非常方便地进行插值。本文将对在学习过程中遇到的四元数的定义、推导、性质、插值等问题进行总结归纳。

复数

我们先来简要讨论一下复数的一些性质以及它与 2D 旋转之间的关系，四元数的很多性质都与复数非常类似，所以理解复数的一些性质对理解四元数非常有帮助。

v′′=z2(z1v)=(z2z1)v\begin{aligned} v'' &= z_{2}(z_{1}v)\\ &= (z_{2}z_{1})v \end{aligned} v′′​=z2​(z1​v)=(z2​z1​)v​

如果我们将这两次旋转所作出的等效变换称为 znetz_{net}znet​，那么

v′′=(z2z1)v=znetvznet=z2z1\begin{aligned} v'' &= (z_{2}z_{1})v = z_{net}v\\ z_{net} &= z_{2}z_{1} \end{aligned} v′′znet​​=(z2​z1​)v=znet​v=z2​z1​​

因为复数的相乘遵守交换律，所以

znet=z2z1=z1z2z_{net} = z_{2}z_{1} = z_{1}z_{2} znet​=z2​z1​=z1​z2​

如果尝试计算一下 znetz_{net}znet​，会发现

znet=(cos⁡(θ)+isin⁡(θ))(cos⁡(ϕ)+isin⁡(ϕ))=cos⁡(θ)cos⁡(ϕ)+i(cos⁡(θ)sin⁡(ϕ))+i(sin⁡(θ)cos⁡(ϕ))−sin⁡(θ)sin⁡(ϕ)=(cos⁡(θ)cos⁡(ϕ)−sin⁡(θ)sin⁡(ϕ))+(cos⁡(θ)sin⁡(ϕ)+sin⁡(θ)cos⁡(ϕ))i\begin{aligned} z_{net} &= (\cos(\theta) + i\sin(\theta))(\cos(\phi) + i\sin(\phi))\\ &= \cos(\theta)\cos(\phi) + i(\cos(\theta)\sin(\phi)) + i(\sin(\theta)\cos(\phi)) - \sin(\theta)\sin(\phi)\\ &= (\cos(\theta)\cos(\phi) - \sin(\theta)\sin(\phi)) + (\cos(\theta)\sin(\phi) + \sin(\theta)\cos(\phi))i \end{aligned} znet​​=(cos(θ)+isin(θ))(cos(ϕ)+isin(ϕ))=cos(θ)cos(ϕ)+i(cos(θ)sin(ϕ))+i(sin(θ)cos(ϕ))−sin(θ)sin(ϕ)=(cos(θ)cos(ϕ)−sin(θ)sin(ϕ))+(cos(θ)sin(ϕ)+sin(θ)cos(ϕ))i​

这个式子可以利用三角恒等式化简为：

znet=(cos⁡(θ)+isin⁡(θ))(cos⁡(ϕ)+isin⁡(ϕ))=cos⁡(θ+ϕ)+isin⁡(θ+ϕ)\begin{aligned} z_{net} &= (\cos(\theta) + i\sin(\theta))(\cos(\phi) + i\sin(\phi))\\ &= \cos(\theta + \phi) + i\sin(\theta + \phi) \end{aligned} znet​​=(cos(θ)+isin(θ))(cos(ϕ)+isin(ϕ))=cos(θ+ϕ)+isin(θ+ϕ)​

所以，当我们对两个 2D 旋转进行复合时，所得到的变换 znetz_{net}znet​ 仍是一个旋转，而且与施加的次序无关。这个等效变换的旋转角是 $z_{1} 与 $z_{2} 旋转角之和。

轴角式旋转

假设我们有一个经过原点的（如果旋转轴不经过原点我们可以先将旋转轴平移到原点，进行旋转，再平移回原处）旋转轴 u⃗=(x,y,z)T\vec{u} = (x,y,z)^Tu=(x,y,z)T，我们希望将一个向量 v⃗\vec{v}v，沿着这个旋转轴旋转 θ\thetaθ 度，变换到 v′⃗\vec{v'}v′

我们使用右手坐标系统，并且将使用右手定则来定义旋转的正方向。你可以将右手拇指指向旋转轴 u⃗\vec{u}u 的正方向，这时其他四个手指弯曲的方向即为旋转的正方向。在上图中即为逆时针方向。

为了消除旋转轴 u⃗\vec{u}u 模长这个多余的自由度，我们可以规定 u⃗\vec{u}u 的模长为 1，即 u⃗\vec{u}u 是一个单位向量。规定 u⃗\vec{u}u 为单位向量后能为我们带来很多便利，这也是数学和物理中对方向定义的惯例。如果 u⃗\vec{u}u 不是一个单位向量，我们可以通过下面的公式将它转化为一个单位向量

u^=u⃗∥u⃗∥\hat{u} = \frac{\vec{u}}{\lVert \vec{u} \rVert} u^=∥u∥u​

旋转的分解

我们可以将 v⃗\vec{v}v 分解为平行于旋转轴 u⃗\vec{u}u 以及正交于 u⃗\vec{u}u 的两个分量，v∥⃗\vec{v_{\parallel}}v∥​​ 和 v⊥⃗\vec{v_{\perp}}v⊥​​，即

v⃗=v∥⃗+v⊥⃗\vec{v} = \vec{v_{\parallel}} + \vec{v_{\perp}} v=v∥​​+v⊥​​

我们可以分别旋转这两个分量，再将它们旋转的结果相加获得旋转后的向量

v′⃗=v∥′⃗+v⊥′⃗\vec{v'} = \vec{v_{\parallel}'} + \vec{v_{\perp}'} v′=v∥′​​+v⊥′​​

下面是分解的示意图：

可以看到，v∥⃗\vec{v_{\parallel}}v∥​​ 其实就是 v⃗\vec{v}v 在 u⃗\vec{u}u 上的正交投影，根据正交投影公式，我们可以得出：

v∥⃗=proju(v⃗)=u⃗⋅v⃗u⃗⋅u⃗u⃗=u⃗⋅v⃗∥u⃗∥2u⃗=(u⃗⋅v⃗)u⃗\begin{aligned} \vec{v_{\parallel}} &= proj_{u}(\vec{v})\\ &= \frac{\vec{u}\cdot\vec{v}}{\vec{u}\cdot\vec{u}}\vec{u}\\ &= \frac{\vec{u}\cdot\vec{v}}{\lVert \vec{u} \rVert ^2}\vec{u}\\ &= (\vec{u}\cdot\vec{v})\vec{u} \end{aligned} v∥​​​=proju​(v)=u⋅uu⋅v​u=∥u∥2u⋅v​u=(u⋅v)u​

因为 v⃗=v∥⃗+v⊥⃗\vec{v} = \vec{v_{\parallel}} + \vec{v_{\perp}}v=v∥​​+v⊥​​，我们可以得到

v⊥⃗=v⃗−v∥⃗=v⃗−(u⃗⋅v⃗)u⃗\begin{aligned} \vec{v_{\perp}} &= \vec{v} - \vec{v_{\parallel}}\\ &= \vec{v} - (\vec{u}\cdot\vec{v})\vec{u} \end{aligned} v⊥​​​=v−v∥​​=v−(u⋅v)u​

v∥⃗\vec{v_{\parallel}}v∥​​ 的旋转

从之前的图示中可以看到，v∥⃗\vec{v_{\parallel}}v∥​​ 其实没有被旋转，仍然与旋转轴 u⃗\vec{u}u 重合，所以

Theorem 4：3D 旋转公式（向量型，平行情况）
当 v∥⃗\vec{v_{\parallel}}v∥​​ 平行于旋转轴 u⃗\vec{u}u 时，旋转 θ\thetaθ 角度之后的 v∥′⃗\vec{v_{\parallel}'}v∥′​​ 为：

v∥′⃗=v∥⃗\vec{v_{\parallel}'} = \vec{v_{\parallel}}v∥′​​=v∥​​

v⊥⃗\vec{v_{\perp}}v⊥​​ 的旋转

接下来，我们需要处理正交于 u⃗\vec{u}u 的 v⊥⃗\vec{v_{\perp}}v⊥​​。因为这两个向量是正交的，这个旋转可以看作是平面内的一个旋转。因为旋转不改变 v⊥⃗\vec{v_{\perp}}v⊥​​ 的长度，所以路径是一个园。下面是这个旋转的示意图，右侧的为俯视图：

现在，3D 的旋转就被我们转化为了 2D 平面上的旋转，由于在这个平面上我们只有一个向量 v⊥⃗\vec{v_{\perp}}v⊥​​，用它来表示一个旋转是不够的，我们还需构造一个同时正交于 u⃗\vec{u}u 和 v⊥⃗\vec{v_{\perp}}v⊥​​ 的向量 w⃗\vec{w}w，这个可以通过叉乘来获得

w⃗=u⃗×v⊥⃗\vec{w} = \vec{u}\times\vec{v_{\perp}} w=u×v⊥​​

注意叉乘的顺序，因为我们使用的是右手坐标系统，按右手定则这个新的向量 w⃗\vec{w}w 指向 v⊥⃗\vec{v_{\perp}}v⊥​​ 逆时针旋转 π/2\pi / 2π/2 后的方向，并且和 v⊥⃗\vec{v_{\perp}}v⊥​​ 一样也处于正交于 u⃗\vec{u}u 的平面内，因为 ∥u⃗∥=1\lVert\vec{u}\rVert = 1∥u∥=1，我们可以发现：

∥w⃗∥=∥u⃗×v⊥⃗∥=∥u⃗∥⋅∥v⊥⃗∥⋅sin⁡(π/2)=∥v⊥⃗∥\begin{aligned} \lVert\vec{w}\rVert &= \lVert\vec{u}\times\vec{v_{\perp}}\rVert\\ &= \lVert\vec{u}\rVert\cdot\lVert\vec{v_{\perp}}\rVert\cdot\sin(\pi/2)\\ &= \lVert\vec{v_{\perp}}\rVert \end{aligned} ∥w∥​=∥u×v⊥​​∥=∥u∥⋅∥v⊥​​∥⋅sin(π/2)=∥v⊥​​∥​

w⃗\vec{w}w 和 v⊥⃗\vec{v_{\perp}}v⊥​​ 的模长相同，且 w⃗\vec{w}w 也位于圆上，我们可以将 w⃗\vec{w}w 和 v⊥⃗\vec{v_{\perp}}v⊥​​ 作为平面内的两个坐标轴，我们现在可以把 v⊥′⃗\vec{v_{\perp}'}v⊥′​​ 投影到 w⃗\vec{w}w 和 v⊥⃗\vec{v_{\perp}}v⊥​​ 上，将其分解为 vv′⃗\vec{v_{v}'}vv′​​ 和 vw′⃗\vec{v_{w}'}vw′​​，利用三角形的知识我们可以得到：

v⊥′⃗=vv′⃗+vw′⃗=cos⁡(θ)v⊥⃗+sin⁡(θ)w⃗=cos⁡(θ)v⊥⃗+sin⁡(θ)(u⃗×v⊥⃗)\begin{aligned} \vec{v_{\perp}'} &= \vec{v_{v}'} + \vec{v_{w}'}\\ &= \cos(\theta)\vec{v_{\perp}} + \sin(\theta)\vec{w}\\ &= \cos(\theta)\vec{v_{\perp}} + \sin(\theta)(\vec{u}\times\vec{v_{\perp}}) \end{aligned} v⊥′​​​=vv′​​+vw′​​=cos(θ)v⊥​​+sin(θ)w=cos(θ)v⊥​​+sin(θ)(u×v⊥​​)​

Theorem 5：3D 旋转公式（向量型，正交情况）
当 v⊥⃗\vec{v_{\perp}}v⊥​​ 正交于旋转轴 u⃗\vec{u}u 时，旋转 θ\thetaθ 角度之后的 v⊥′⃗\vec{v_{\perp}'}v⊥′​​ 为：

v⊥′⃗=cos⁡(θ)v⊥⃗+sin⁡(θ)(u⃗×v⊥⃗)\vec{v_{\perp}'} = \cos(\theta)\vec{v_{\perp}} + \sin(\theta)(\vec{u}\times\vec{v_{\perp}})v⊥′​​=cos(θ)v⊥​​+sin(θ)(u×v⊥​​)

v⃗\vec{v}v 的旋转

将上面两个结果组合可以获得：

v′⃗=v∥′⃗+v⊥′⃗=v∥⃗+cos⁡(θ)v⊥⃗+sin⁡(θ)(u⃗×v⊥⃗)\begin{aligned} \vec{v'} &= \vec{v_{\parallel}'} + \vec{v_{\perp}'}\\ &= \vec{v_{\parallel}} + \cos(\theta)\vec{v_{\perp}} + \sin(\theta)(\vec{u}\times\vec{v_{\perp}}) \end{aligned} v′​=v∥′​​+v⊥′​​=v∥​​+cos(θ)v⊥​​+sin(θ)(u×v⊥​​)​

因为叉乘遵守分配律，

u⃗×v⊥⃗=u⃗×(v⃗−v∥⃗)=u⃗×v⃗−u⃗×v∥⃗=u⃗×v⃗(u⃗平行于v∥⃗，所以u⃗×v∥⃗=0⃗)\begin{aligned} \vec{u}\times\vec{v_{\perp}} &= \vec{u}\times(\vec{v} - \vec{v_{\parallel}})\\ &= \vec{u}\times\vec{v} - \vec{u}\times\vec{v_{\parallel}}\\ &= \vec{u}\times\vec{v}\quad (\vec{u} 平行于 \vec{v_{\parallel}}，所以 \vec{u}\times\vec{v_{\parallel}} = \vec{0}) \end{aligned} u×v⊥​​​=u×(v−v∥​​)=u×v−u×v∥​​=u×v(u平行于v∥​​，所以u×v∥​​=0)​

最后，将 v∥⃗=(u⃗⋅v⃗)u⃗\vec{v_{\parallel}} = (\vec{u}\cdot\vec{v})\vec{u}v∥​​=(u⋅v)u 和 v⊥⃗=v⃗−(u⃗⋅v⃗)u⃗\vec{v_{\perp}} = \vec{v} - (\vec{u}\cdot\vec{v})\vec{u}v⊥​​=v−(u⋅v)u 代入

v′⃗=(u⃗⋅v⃗)u⃗+cos⁡(θ)(v⃗−(u⃗⋅v⃗)u⃗)+sin⁡(θ)(u⃗×v⃗)=cos⁡(θ)v⃗+(1−cos⁡(θ))(u⃗⋅v⃗)u⃗+sin⁡(θ)(u⃗×v⃗)\begin{aligned} \vec{v'} &= (\vec{u}\cdot\vec{v})\vec{u} + \cos(\theta)(\vec{v} - (\vec{u}\cdot\vec{v})\vec{u}) + \sin(\theta)(\vec{u}\times\vec{v})\\ &= \cos(\theta)\vec{v} + (1 - \cos(\theta))(\vec{u}\cdot\vec{v})\vec{u} + \sin(\theta)(\vec{u}\times\vec{v}) \end{aligned} v′​=(u⋅v)u+cos(θ)(v−(u⋅v)u)+sin(θ)(u×v)=cos(θ)v+(1−cos(θ))(u⋅v)u+sin(θ)(u×v)​

这样我们就得到了一般形式的旋转公式：

Theorem 6：3D 旋转公式（向量型，一般情况）
3D 空间中任意一个 v⃗\vec{v}v 沿着单位向量 u⃗\vec{u}u 旋转 θ\thetaθ 角度之后的 v⊥′⃗\vec{v_{\perp}'}v⊥′​​ 为：

v′⃗=cos⁡(θ)v⃗+(1−cos⁡(θ))(u⃗⋅v⃗)u⃗+sin⁡(θ)(u⃗×v⃗)\vec{v'} = \cos(\theta)\vec{v} + (1 - \cos(\theta))(\vec{u}\cdot\vec{v})\vec{u} + \sin(\theta)(\vec{u}\times\vec{v})v′=cos(θ)v+(1−cos(θ))(u⋅v)u+sin(θ)(u×v)
